Page methods

Use these methods with a Page object.

Aliases 

PAGE.Aliases ⟼ []string
Returns the URL aliases as defined in front matter. Read More »

AllTranslations 

PAGE.AllTranslations ⟼ page.Pages
Returns all translations of the given page, including the current language. Read More »

AlternativeOutputFormats 

PAGE.AlternativeOutputFormats ⟼ page.OutputFormats
Returns a slice of OutputFormat objects, excluding the current output format, each representing one of the output formats enabled for the given page. Read More »

Ancestors 

PAGE.Ancestors ⟼ page.Pages
Returns a collection of Page objects, one for each ancestor section of the given page. Read More »

BundleType 

PAGE.BundleType ⟼ string
Returns the bundle type of the given page, or an empty string if the page is not a page bundle. Read More »

CodeOwners 

PAGE.CodeOwners ⟼ []string
Returns of slice of code owners for the given page, derived from the CODEOWNERS file in the root of the project directory. Read More »

Content 

PAGE.Content ⟼ template.HTML
Returns the rendered content of the given page. Read More »

CurrentSection 

PAGE.CurrentSection ⟼ page.Page
Returns the Page object of the section in which the given page resides. Read More »

Data 

PAGE.Data ⟼ page.Data
Returns a unique data object for each page kind. Read More »

Date 

PAGE.Date ⟼ time.Time
Returns the date of the given page. Read More »

Description 

PAGE.Description ⟼ string
Returns the description of the given page as defined in front matter. Read More »

Draft 

PAGE.Draft ⟼ bool
Reports whether the given page is a draft as defined in front matter. Read More »

Eq 

PAGE1.Eq PAGE2 ⟼ bool
Reports whether two Page objects are equal. Read More »

ExpiryDate 

PAGE.ExpiryDate ⟼ time.Time
Returns the expiry date of the given page. Read More »

File 

PAGE.File ⟼ hugolib.fileInfo
For pages backed by a file, returns file information for the given page. Read More »

FirstSection 

PAGE.FirstSection ⟼ page.Page
Returns the Page object of the top level section of which the given page is a descendant. Read More »

Fragments 

PAGE.Fragments ⟼ tableofcontents.Fragments
Returns a data structure of the fragments in the given page. Read More »

FuzzyWordCount 

PAGE.FuzzyWordCount ⟼ int
Returns the number of words in the content of the given page, rounded up to the nearest multiple of 100. Read More »

GetPage 

PAGE.GetPage PATH ⟼ page.Page
Returns a Page object from the given path. Read More »

GetTerms 

PAGE.GetTerms TAXONOMY ⟼ page.Pages
Returns a collection of term pages for terms defined on the given page in the given taxonomy, ordered according to the sequence in which they appear in front matter. Read More »

GitInfo 

PAGE.GitInfo ⟼ source.GitInfo
Returns Git information related to the last commit of the given page. Read More »

HasMenuCurrent 

PAGE.HasMenuCurrent MENU MENUENTRY ⟼ bool
Reports whether the given Page object matches the Page object associated with one of the child menu entries under the given menu entry in the given menu. Read More »

HasShortcode 

PAGE.HasShortcode NAME ⟼ bool
Reports whether the given shortcode is called by the given page. Read More »

HeadingsFiltered 

PAGE.HeadingsFiltered ⟼ tableofcontents.Headings
Returns a slice of headings for each page related to the given page. Read More »

InSection 

PAGE.InSection SECTION ⟼ bool
Reports whether the given page is in the given section. Read More »

IsAncestor 

PAGE1.IsAncestor PAGE2 ⟼ bool
Reports whether PAGE1 is an ancestor of PAGE2. Read More »

IsDescendant 

PAGE1.IsDescendant PAGE2 ⟼ bool
Reports whether PAGE1 is a descendant of PAGE2. Read More »

IsHome 

PAGE.IsHome ⟼ bool
Reports whether the given page is the home page. Read More »

IsMenuCurrent 

PAGE.IsMenuCurrent MENU MENUENTRY ⟼ bool
Reports whether the given Page object matches the Page object associated with the given menu entry in the given menu. Read More »

IsNode 

PAGE.IsNode ⟼ bool
Reports whether the given page is a node. Read More »

IsPage 

PAGE.IsPage ⟼ bool
Reports whether the given page is a regular page. Read More »

IsSection 

PAGE.IsSection ⟼ bool
Reports whether the given page is a section page. Read More »

IsTranslated 

PAGE.IsTranslated ⟼ bool
Reports whether the given page has one or more translations. Read More »

Keywords 

PAGE.Keywords ⟼ []string
Returns a slice of keywords as defined in front matter. Read More »

Kind 

PAGE.Kind ⟼ string
Returns the kind of the given page. Read More »

Language 

PAGE.Language ⟼ langs.Language
Returns the language object for the given page. Read More »

Lastmod 

PAGE.Lastmod ⟼ time.Time
Returns the last modification date of the given page. Read More »

Layout 

PAGE.Layout ⟼ string
Returns the layout for the given page as defined in front matter. Read More »

Len 

PAGE.Len ⟼ int
Returns the length, in bytes, of the rendered content of the given page. Read More »

LinkTitle 

PAGE.LinkTitle ⟼ string
Returns the link title of the given page. Read More »

Next 

PAGE.Next ⟼ page.Page
Returns the next page in a global page collection, relative to the given page. Read More »

NextInSection 

PAGE.NextInSection ⟼ page.Page
Returns the next page within a section, relative to the given page. Read More »

OutputFormats 

PAGE.OutputFormats ⟼ []OutputFormat
Returns a slice of OutputFormat objects, each representing one of the output formats enabled for the given page. Read More »

Page 

PAGE.Page ⟼ page.Page
Returns the Page object of the given page. Read More »

Pages 

PAGE.Pages ⟼ page.Pages
Returns a collection of regular pages within the current section, and section pages of immediate descendant sections. Read More »

Paginate 

PAGE.Paginate COLLECTION [N] ⟼ page.Pager
Paginates a collection of pages. Read More »

Paginator 

PAGE.Paginator ⟼ page.Pager
Paginates the collection of regular pages received in context. Read More »

Param 

PAGE.Param KEY ⟼ any
Returns a page parameter with the given key, falling back to a site parameter if present. Read More »

Params 

PAGE.Params ⟼ maps.Params
Returns a map of custom parameters as defined in the front matter of the given page. Read More »

Parent 

PAGE.Parent ⟼ page.Page
Returns the Page object of the parent section of the given page. Read More »

Path 

PAGE.Path ⟼ string
Returns the logical path of the given page. Read More »

Permalink 

PAGE.Permalink ⟼ string
Returns the permalink of the given page. Read More »

Plain 

PAGE.Plain ⟼ string
Returns the rendered content of the given page, removing all HTML tags. Read More »

PlainWords 

PAGE.PlainWords ⟼ []string
Calls the Plain method, splits the result into a slice of words, and returns the slice. Read More »

Prev 

PAGE.Prev ⟼ page.Page
Returns the previous page in a global page collection, relative to the given page. Read More »

PrevInSection 

PAGE.PrevInSection ⟼ page.Page
Returns the previous page within a section, relative to the given page. Read More »

PublishDate 

PAGE.PublishDate ⟼ time.Time
Returns the publish date of the given page. Read More »

RawContent 

PAGE.RawContent ⟼ string
Returns the raw content of the given page. Read More »

ReadingTime 

PAGE.ReadingTime ⟼ int
Returns the estimated reading time, in minutes, for the given page. Read More »

Ref 

PAGE.Ref OPTIONS ⟼ string
Returns the absolute URL of the page with the given path, language, and output format. Read More »

RegularPages 

PAGE.RegularPages ⟼ page.Pages
Returns a collection of regular pages within the current section. Read More »

RegularPagesRecursive 

PAGE.RegularPagesRecursive ⟼ page.Pages
Returns a collection of regular pages within the current section, and regular pages within all descendant sections. Read More »

RelPermalink 

PAGE.RelPermalink ⟼ string
Returns the relative permalink of the given page. Read More »

RelRef 

PAGE.RelRef OPTIONS ⟼ string
Returns the relative URL of the page with the given path, language, and output format. Read More »

Render 

PAGE.Render NAME ⟼ template.HTML
Renders the given template with the given page as context. Read More »

RenderShortcodes 

PAGE.RenderShortcodes ⟼ template.HTML
Renders all shortcodes in the content of the given page, preserving the surrounding markup. Read More »

RenderString 

PAGE.RenderString [OPTIONS] MARKUP ⟼ template.HTML
Renders markup to HTML. Read More »

Resources 

PAGE.Resources ⟼ resource.Resources
Returns a collection of page resources. Read More »

Scratch 

PAGE.Scratch ⟼ maps.Scratch
Returns a “scratch pad” on the given page to store and manipulate data. Read More »

Section 

PAGE.Section ⟼ string
Returns the name of the top level section in which the given page resides. Read More »

Sections 

PAGE.Sections ⟼ page.Pages
Returns a collection of section pages, one for each immediate descendant section of the given page. Read More »

Site 

PAGE.Site ⟼ page.siteWrapper
Returns the Site object. Read More »

Sitemap 

PAGE.Sitemap ⟼ config.SitemapConfig
Returns the sitemap settings for the given page as defined in front matter, falling back to the sitemap settings as defined in the site configuration. Read More »

Sites 

PAGE.Sites ⟼ page.Sites
Returns a collection of all Site objects, one for each language, ordered by language weight. Read More »

Slug 

PAGE.Slug ⟼ string
Returns the URL slug of the given page as defined in front matter. Read More »

Store 

PAGE.Store ⟼ maps.Scratch
Returns a persistent “scratch pad” on the given page to store and manipulate data. Read More »

Summary 

PAGE.Summary ⟼ template.HTML
Returns the content summary of the given page. Read More »

TableOfContents 

PAGE.TableOfContents ⟼ template.HTML
Returns a table of contents for the given page. Read More »

Title 

PAGE.Title ⟼ string
Returns the title of the given page. Read More »

TranslationKey 

PAGE.TranslationKey ⟼ string
Returns the translation key of the given page. Read More »

Translations 

PAGE.Translations ⟼ page.Pages
Returns all translations of the given page, excluding the current language. Read More »

Truncated 

PAGE.Truncated ⟼ bool
Reports whether the content length exceeds the summary length. Read More »

Type 

PAGE.Type ⟼ string
Returns the content type of the given page. Read More »

Weight 

PAGE.Weight ⟼ int
Returns the weight of the given page as defined in front matter. Read More »

WordCount 

PAGE.WordCount ⟼ int
Returns the number of words in the content of the given page. Read More »
