RelPermalink

Returns the relative permalink of the given page.

Syntax

PAGE.RelPermalink

Returns

string

Site configuration:

hugo.
      
baseURL: https://example.org/docs/
title: Documentation

baseURL = 'https://example.org/docs/'
title = 'Documentation'

{
   "baseURL": "https://example.org/docs/",
   "title": "Documentation"
}

Template:

{{ $page := .Site.GetPage "/about" }}
{{ $page.RelPermalink }} → /docs/about/
